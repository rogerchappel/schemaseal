# Roadmap

## Near term

- Expand JSON Schema support beyond the MVP subset.
- Add `schemaseal drift` as a focused command for pins-only checks.
- Add baseline comparison reports for before/after PR evidence.
- Add fixture generation helpers for common config formats.

## Later

- Optional SARIF output for CI annotations.
- Optional policy files for fail thresholds per path.
- Better Markdown/text schema affordances.
- Release packaging polish for npm and Homebrew.

## Non-goals

- Hosted dashboard.
- Telemetry in CLI runtime.
- Automatic network schema registry fetching.
